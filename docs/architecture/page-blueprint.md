## / — Top Page

**Purpose:** ENISHIとは何か・何を作っているかを5秒で伝える

### Sections

```
┌──────────────────────────────────────────┐
│ 1. HERO                                  │
│    H1: メインコピー（ENISHIのwhy）        │
│    Sub: サブコピー（what / how）          │
│    CTA1: プロジェクトを見る → /projects  │
│    CTA2: お問い合わせ → /contact         │
│    Visual: Hero image / animation         │
├──────────────────────────────────────────┤
│ 2. FEATURED PROJECTS                     │
│    Title: "Projects"                     │
│    Cards: featured=true のプロジェクト   │
│    Link: すべてのプロジェクト → /projects│
├──────────────────────────────────────────┤
│ 3. VISION / PHILOSOPHY                   │
│    ENISHIが目指す世界観                   │
│    "縁" の意味・なぜこの名前か           │
│    3つの価値観（短文）                   │
├──────────────────────────────────────────┤
│ 4. ABOUT SNIPPET                         │
│    会社・創業者の1行紹介                  │
│    Link: 詳しく知る → /about             │
│    Image: 創業者 / チーム                 │
├──────────────────────────────────────────┤
│ 5. CONTACT CTA                           │
│    "縁をつなぎましょう" 等のコピー        │
│    CTA: お問い合わせ → /contact          │
└──────────────────────────────────────────┘
```

| Section | Priority | コンテンツ責任者 |
|---------|---------|----------------|
| Hero | 必須 | 創業者 |
| Featured Projects | 必須 | Content Collections自動 |
| Vision | 必須 | 創業者 |
| About Snippet | 推奨 | 創業者 |
| Contact CTA | 必須 | デザイン固定テキスト |

## /contact — Contact

**Purpose:** 関心ある訪問者を具体的な問い合わせに変換する

### Sections

```
┌──────────────────────────────────────────┐
│ 1. HERO                                  │
│    H1: "Contact" / "お問い合わせ"         │
│    Sub: 誰からの問い合わせを歓迎するか   │
│         (顧客・プレス・候補者・一般)     │
├──────────────────────────────────────────┤
│ 2. CONTACT FORM                          │
│    Fields:                               │
│      - お名前 (text, required)           │
│      - メールアドレス (email, required)  │
│      - お問い合わせ種別 (select):        │
│          ビジネス / プレス / 採用 / その他 │
│      - メッセージ (textarea, required)   │
│    Submit: "送信する"                    │
│    Response: "通常3営業日以内にご返信します" │
├──────────────────────────────────────────┤
│ 3. ALTERNATIVES                          │
│    メール直接 / SNSリンク               │
└──────────────────────────────────────────┘
```

**Contact Form実装注記:**
- MVP: Netlify Forms / Formspree 等の静的フォームサービス
  または mailto: リンク（最小実装）
- Growth: API Route + メール送信サービス (Resend等)
