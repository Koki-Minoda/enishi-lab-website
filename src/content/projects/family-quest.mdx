## Overview

家族みんなで楽しめるタスク・クエスト管理アプリです。
子育てをゲームに変え、家族の絆を深めます。

## Problem

家族のタスク管理は義務的になりがちです。
親子間のコミュニケーションが減少している現代において、
「楽しさ」を通じて家族をつなぐ仕組みが求められています。

## Solution

クエスト形式でタスクをゲーム化。子供は冒険者として、
親はクエストを設定するゲームマスターとして参加します。
完了ごとにポイントや報酬を設定でき、達成感が次の行動へつながります。

## Key Features

- クエスト作成: 家事・勉強などをゲームのミッションとして設定
- ファミリーボード: 家族全員の進捗をリアルタイムで確認
- 報酬システム: 達成したらポイントやご褒美を設定可能
- 振り返り: 週次レポートで家族の活動を振り返る
