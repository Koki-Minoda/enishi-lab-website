## Overview

大切な縁を記録し、関係を育てるためのノートアプリです。
ENISHIという名前の由来であり、原点となるプロダクトです。

## Problem

出会った人や大切にしたい縁は、時間とともに薄れてしまいがちです。
SNSのフォローとは違う、本質的な「縁の管理」の仕組みが存在しません。

## Solution

シンプルで温かみのあるUIで、人との縁を記録・育てます。
誕生日・記念日の通知、エピソードの記録、関係の深さの可視化などで
大切な縁を能動的に育てるサポートをします。

## Key Features

- 縁の記録: 出会い・エピソード・感謝をノートに残す
- リマインダー: 大切な日を忘れないための通知
- 縁の深度: 関係の深さを可視化するグラフ
- プライベート設計: データはすべてデバイス内に保存

## Current Status

コンセプト設計・開発中。
