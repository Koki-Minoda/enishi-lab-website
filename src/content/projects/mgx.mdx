## Overview

MGXは、ポーカープレイヤーの思考力と戦略性を高めるためのプラットフォームです。
AI対戦・ハンド分析・トレーニングモードを通じて、実力を体系的に伸ばせます。

## Problem

ポーカーの上達は孤独で非効率なことが多いです。
フィードバックが得られない反復練習では、悪習慣が固定されるリスクがあります。

## Solution

AIが対戦相手となり、一手ごとにフィードバックを提供。
弱点を可視化し、改善サイクルを最速化します。

## Key Features

- AI対戦: 様々なスタイルのAI相手に対戦練習
- ハンド分析: プレイの振り返りと最適解の確認
- トレーニング: 特定状況に特化したドリルモード
- 統計ダッシュボード: 長期的な成長トラッキング

## Current Status

Badugiを起点に開発中。将来的には複数のポーカー種目へ展開予定。
